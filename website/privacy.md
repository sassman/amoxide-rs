# Privacy Policy

**Last updated:** March 2026

## No Personal Data Collected

This website does not collect, store, or process any personal data. There are no cookies, no sign-up forms, no tracking pixels, and no third-party advertising.

## Analytics

We use [Plausible Analytics](https://plausible.io) to understand how the site is used. Plausible is a privacy-friendly analytics tool that:

- Does **not** use cookies
- Does **not** collect personal data
- Does **not** track users across sites
- Is fully compliant with GDPR, CCPA, and PECR
- Stores no data in the visitor's browser

All data is aggregated — we see page view counts and referral sources, not individual visitors. Plausible's data processing is covered by their [data policy](https://plausible.io/data-policy).

## Open Source

This website is open source. You can review the full source code at [github.com/sassman/amoxide-rs](https://github.com/sassman/amoxide-rs).
